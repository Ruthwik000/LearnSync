Build a full-featured frontend web application called **LearnSync** using **React (functional components, hooks)** and **Tailwind CSS**.

LearnSync is an AI-powered learning coordination platform designed for NGOs to manage students, mentors, and educational programs.

The system must simulate backend behavior using **localStorage + mock data**, while maintaining real-world workflows and a clean SaaS-style UI.

---

# 🧠 CORE PRODUCT IDEA

LearnSync is a connected ecosystem where:

* Students receive personalized, structured learning
* Mentors teach, create content, and support students
* NGOs monitor performance and impact
* AI enhances learning, planning, and decision-making

---

# 🏗️ SYSTEM ARCHITECTURE

There are 4 main layers:

1. Student Interface (Learning + Gamification)
2. Mentor Interface (Teaching + Content Creation)
3. NGO Admin Interface (Analytics + Control)
4. AI Layer (Gemini 2.5 Flash integration)

---

# 🎨 DESIGN SYSTEM (STRICT RULES)

* NO gradients
* NO flashy UI
* NO glassmorphism

Use:

* `bg-gray-50` (background)
* `bg-white` (cards)
* `border border-gray-200`
* `text-gray-900` (primary text)
* `text-gray-500` (secondary text)
* `bg-blue-600 text-white` (primary buttons)
* `text-green-600` (success)
* `text-red-500` (alerts)

UI style must feel like:
👉 Notion / Stripe / Linear

Use:

* `rounded-xl`
* `p-4`, `p-6`
* `gap-4`
* `shadow-sm`
