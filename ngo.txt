Build a complete **NGO Admin Portal** for LearnSync using **React (functional components + hooks)** and **Tailwind CSS**.

The NGO Portal is the **control + analytics layer** of the system where administrators:

* Monitor students and mentors
* Track learning outcomes
* Analyze performance and attendance
* Take data-driven decisions

All data must be handled using **localStorage (mock data)**.

UI must be **clean, minimal, professional (NO gradients)**.

---

# 🧠 CORE OBJECTIVE

The NGO Admin Portal must:

* Provide complete visibility into learning activities
* Measure student improvement and engagement
* Track mentor performance
* Identify risks and weak areas
* Enable data-driven decisions
