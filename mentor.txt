Build a complete **Mentor Portal** for LearnSync using **React (functional components + hooks)** and **Tailwind CSS**.

The Mentor Portal represents the **execution layer** of the system where mentors:

* Teach students
* Create learning content
* Respond to doubts
* Track student progress

All data must be handled using **localStorage (mock data)**.

UI must be **clean, minimal, professional (NO gradients)**.

---

# 🧠 CORE OBJECTIVE

The Mentor Portal must:

* Enable mentors to create structured learning content
* Automatically convert content into student-ready modules (via AI)
* Allow session tracking and updates
* Provide visibility into student performance
* Enable collaborative doubt resolution

---

# 📅 6. ATTENDANCE MANAGEMENT SYSTEM (CRITICAL FEATURE)

---

## 🎯 Goal:

Allow mentors to **track student attendance for every session**, enabling NGOs to monitor engagement and consistency.

---

## 🧩 UI: Attendance Panel (Per Session)

When mentor starts or logs a session:

### Show Student List:

| Student Name | Present | Absent |
| ------------ | ------- | ------ |

---

## Interaction:

* Toggle buttons:

  * ✅ Present
  * ❌ Absent

---

## Default:

* All students marked “Present” (editable)

---

## Button:

👉 “Save Attendance”

---

## 💾 Store:

```js
{
  sessionId,
  date: "2026-03-30",
  attendance: [
    { studentId: 1, status: "present" },
    { studentId: 2, status: "absent" }
  ]
}
```

---

## 🔗 INTEGRATION WITH SESSION

Attendance should be:

* Linked with session logging
* Saved together with session data

---

## 📊 IMPACT ON SYSTEM

### 1. Student Side:

* Show attendance % in dashboard

---

### 2. Mentor Side:

* Show:

  * Attendance rate per student
  * Frequent absentees

---

### 3. NGO Dashboard:

* Track:

  * Overall attendance %
  * Drop in engagement
  * Inactive students

---

## ⚠️ ALERTS (IMPORTANT)

Trigger alerts:

* “Student absent for 3 sessions”
* “Attendance below 50%”

---

## 🎯 UX DESIGN

* Use simple toggle buttons
* Color coding:

  * Green → Present
  * Red → Absent

---

## 🏆 FINAL GOAL

Attendance system should:
👉 Make learning measurable
👉 Help NGOs track real engagement
👉 Improve accountability
